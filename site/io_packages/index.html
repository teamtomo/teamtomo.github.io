    Input / Output Packages#

    Analysis of cryo-EM or cryo-ET data often involves a little scripting, these file input and output packages are there to make that a little easier.

    Metadata#

    • starfile | read and write STAR files |
    • imodmodel | read and write IMOD model files |
    • mdocfile | read and write SerialEM metadata files |
    • alnfile | read AreTomo alignment files |
    • etomofiles | read IMOD etomo alignment files |
    • dynamotable | read and write Dynamo table files |

    Images & Models#

    • eerfile | read electron event representation files |
    • emfile | read and write EM image files |
    • mmdf | read and write macromolecular structure files |

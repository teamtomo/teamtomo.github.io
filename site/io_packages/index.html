    Input / Output Packages#

    Cryo-EM/ET software often read and write domain specific file formats.

    Analysis of cryo-ET data often involves a little scripting, these file input and output packages are there to make that a little easier.

    • starfile

      Read and write relion and other starfiles

    • imodmodel

      Read and write imod model files

    • mdocfile

      Read and write SerialEM metadata files

    • emfile

      Read and write EM image files

    • dynamotable

      Read and write Dynamo table files

    • mmdf

      Read and write macromolecular structure files
